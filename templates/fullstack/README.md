# Rackr project scaffold

### Running

1. `$ bundle install`
2. `$ yarn`
3. `$ bin/app/assets`
4. `$ bin/app/run`

### Migrations

- `$ db/migrate`

### Included

gems:
- `rackr` for routes
- `sequel` + `sqlite3` for database work
- `rerun` for reload after changes
- `rspec` for tests

js:
- `@hotwired/turbo`

assets:
- `esbuild` for bundle js assets
